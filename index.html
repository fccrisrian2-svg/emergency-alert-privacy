Política de Privacidad - Aviso Familia

Última actualización: marzo 2026

1. ¿Qué es Aviso Familia?

Es una app de comunicación entre personas de confianza para enviar avisos de necesidad de forma simple.

2. ¿Qué información usamos?

Recopilamos: Nombre, número de teléfono o correo para identificarte, el contacto que designes para avisos e identificadores de dispositivo para notificaciones.

3. Sin seguimiento de ubicación

Esta aplicación no recopila, no accede ni comparte tu ubicación geográfica.

4. Uso de datos

Los datos se usan para enviar avisos, autenticar tu identidad y enviar notificaciones vía Firebase. No vendemos datos a terceros.

5. Eliminación de datos

Podés solicitar la eliminación de tu cuenta y datos escribiendo a: fccrisrian2@gmail.com.
